flat
====

Rename files under subdirectories to flat names

Original file names:

    ./icon/ic_a.png
    ./icon/ic_b.png
    ./map/map_a.png
    ./map/map_b.png
    ./map/asia/korea.png
    ./map/asia/japan.png

New file name:

    ./icon__ic_a.png
    ./icon__ic_b.png
    ./map__map_a.png
    ./map__map_b.png
    ./map__asia__korea.png
    ./map__asia__japan.png
